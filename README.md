# Revo Capital — marketing site

Vite + React + TypeScript + Tailwind. Client-side routes via React Router.

## Setup

```bash
npm install
```

Optional: copy `.env.example` to `.env` if you add env-based config later.

## Scripts

| Command         | Description              |
|----------------|--------------------------|
| `npm run dev`  | Local dev server         |
| `npm run build`| Production build → `dist`|
| `npm run preview` | Preview production build |
| `npm run lint` | `tsc --noEmit`           |

Dev server defaults to port **3000** (see `vite.config.ts`).
